# 一、世界观定位

## 核心定位

《灵脉守御》不是传统仙侠塔防。

而是：

> “以凡人流修仙生态为基础的灵脉经营 + 阵法构筑 + 妖潮防御 Roguelike 修仙塔防游戏。”

核心关键词：

- 凡人流
- 苟道修仙
- 灵脉经营
- 阵法自动战斗
- 法宝构筑
- 五行反应
- 长线成长
- 飞升体系

---

# 二、世界背景

天地分为：

```text
人界
灵界
魔界
真仙界
```

玩家作为低阶散修，从凡俗界开始修炼。

通过：

- 开辟灵脉
- 建造阵法
- 炼制法宝
- 培育灵兽
- 对抗妖潮
- 渡劫突破

最终飞升灵界乃至仙界。

---

# 三、核心世界规则

## 1. 灵气资源有限

世界并非无限成长。

核心矛盾：

```text
资源稀缺
灵气争夺
寿元限制
```

因此玩家必须：

- 经营灵脉
- 合理构筑
- 管理资源
- 规划成长路线

---

## 2. 凡人流修仙生态

游戏强调：

- 小心成长
- 风险控制
- 资源积累
- Build 联动

而不是：

- 无脑割草
- 无限数值膨胀

---

## 3. 飞升体系

玩家目标：

```text
炼气 → 筑基 → 结丹 → 元婴 → 化神 → 炼虚 → 合体 → 大乘 → 飞升
```

飞升后解锁：

- 灵界玩法
- 真灵体系
- 法则系统
- 时间法则
- 仙界阵法

# 五、灵脉系统

## 核心定位

灵脉是整个游戏的资源核心。

玩家需要：

- 占领灵脉节点
- 连接灵气网络
- 建造聚灵阵
- 维持灵气流动

---

## 灵脉等级

```text

玄天灵脉

```
