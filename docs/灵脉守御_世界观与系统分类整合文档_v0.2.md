# 《灵脉守御》世界观与系统分类整合文档（更新版）

版本：v0.2

---

# 一、项目核心定位

## 项目类型

- 修仙题材 Roguelike 塔防
- Survivor-like + Build 构筑
- 自动战斗 + 阵法经营
- Steam 单机买断制

---

# 二、核心世界观

天地灵脉正在崩塌。

修士必须争夺灵脉节点才能继续修行。

因此形成：

- 洞府争夺
- 阵法战争
- 妖潮侵袭
- 灵气战争
- 禁术复苏

# 十三、设计原则

## 高频成长

玩家每30~60秒获得成长反馈。

## 强差异化

核心必须是：

阵法构筑 + 灵脉经营

而不是简单修仙皮。
