# 《凡人修仙传》六系大百科


## 一、法宝篇

### 1.1 法宝等级体系

法宝是修士搏命的核心依仗。在小说设定中，法宝按品阶从低到高可分为多个层级，与人界、灵界、仙界三个阶段相对应：

| 层级 | 品阶名称 | 适用境界 | 核心特征 |
| :--- | :--- | :--- | :--- |
| 1 | **法器** | 炼气·筑基期 | 分下、中、上、顶级四档，制作和使用门槛低，是低阶修士标配 |
| 2 | **符宝/真宝** | 跨阶通用（消耗品） | 结丹以上修士将法宝威能封印于符箓中，符宝可多次使用，最多封印十分之一威能；真宝为顶级符宝，可封印三分之一威能，但只能血脉后人使用且为一次性 |
| 3 | **法宝** | 结丹期及以上 | 结丹修士以丹火培炼的本命之物，威力随主人修为提升而增强 |
| 4 | **本命法宝** | 结丹期及以上 | 与修士心神融合，法宝受损修士也会受伤，不可被轻易夺取 |
| 5 | **古宝** | 元婴期及以上 | 上古修士遗留的法宝，威力通常远胜同阶，但大多无配套通宝诀 |
| 6 | **仿通天灵宝** | 元婴期及以上 | 模仿通天灵宝炼制而成，威力不及正品，但在人界已属顶级。如韩立的三焰扇 |
| 7 | **通天灵宝** | 元婴后期及以上 | 上古流传，每一件均有专属通宝诀，可封印体内培炼，具翻江倒海之威。通宝诀是区分通天灵宝与普通古宝的关键 |
| 8 | **玄天圣器** | 大乘期 | 灵界一方霸主或大族的镇族之宝 |
| 9 | **玄天残宝** | 大乘期 | 玄天之宝的残损品，威能远超通天灵宝 |
| 10 | **玄天之宝** | 大乘期及以上 | 天地自然孕育而成，每一界不过十几件，蕴含法则之力，可沟通上古真灵 |
| 11 | **仙器** | 真仙及以上（仙界） | 分九品，一品最强；韩立的青竹蜂云剑后期晋升为三品仙器 |

**补充说明**：掌天瓶为天道初显时诞生的第一件玄天之物，蕴含时间法则之力，可凝聚绿液催熟灵药，完整形态可凝聚时间晶粒穿越时空。严格来说其品阶已超越普通玄天之宝范畴，属于混沌至宝级别。

### 1.2 法宝类型划分

除按品阶分层外，法宝还可按功能与获取途径划分：

*   **按功能划分**：攻击类（飞剑、飞刀等）、防御类（盾牌、护罩等）、辅助类（飞行、储物等）、特殊类（封印、困敌等）。
*   **按获取途径划分**：炼制法宝（以自身丹火培炼的本命法宝）、古宝（上古遗留）、得宝炼化（获得天地异宝后炼化所得）。
*   **按战斗定位划分**：主战法宝（日常对敌主力）、底牌法宝（越阶击杀的关键）、功能法宝（辅助生存与跑路）。

## 二、灵丹篇

### 2.1 灵丹体系概览

在《凡人修仙传》中，丹药是决定修士修行速度与能否突破瓶颈的命脉所在。修仙世界的残酷竞争，很大程度上体现在丹药资源的争夺上。灵丹可按功能分为以下几大类：

*   **修炼丹**：精进修为，加速法力积累（如炼气散、培元丹、高阶的合气丹等）
*   **突破丹（破境丹）**：辅助冲关破镜，是最稀缺、争夺最激烈的丹药类型
    - **筑基丹**：炼气突破至筑基的关键丹药，天南修仙界每十年各派为争夺筑基丹名额而激烈角逐。人界中，筑基丹的核心材料多来自血色禁地等秘境，灵药种子极为珍贵
    - **结金丹/凝丹散**：辅助筑基大圆满凝结金丹
    - **化婴丹/凝婴丹**：辅助金丹大圆满碎丹化婴
    - **化神丹**：辅助元婴后期冲击化神期的稀世丹药
    - **培婴丹**：元婴期进阶所用，需阴芝马等稀有灵药炼制
    - **补天丹**：可补全先天灵根缺陷的逆天丹药，封存于虚天鼎中
*   **疗伤丹与解毒丹**：恢复伤势、解毒保命
*   **延寿丹**：增加寿元，每多活一年就多一分突破的可能
*   **其他特殊丹药**：如激发潜力的燃元丹、临时提升神识的丹药等

### 2.2 掌天瓶与丹药体系的关系

丹药的炼制离不开灵药，而年份足够的灵药在修仙界极为稀缺。掌天瓶作为韩立的核心依仗，其产生的**参天造化露**可以无视法则去增加灵药的年份，只要有根茎或种子即可培育出万年药龄的灵药。这意味着对韩立而言，丹药炼制的基础材料几乎是无限的——这也是他能以四属性“伪灵根”资质逆天改命、一路突破的核心原因之一。

## 三、妖兽篇

### 3.1 妖兽等级体系

妖兽的等级划分与修士境界存在明确对应关系：

| 妖兽等级 | 对应修士境界 | 核心特征 |
| :--- | :--- | :--- |
| 1级 | 炼气期 | 初开灵智，不会言语，能使用基础妖术 |
| 2级 | 筑基初期 | 开始具备基础法术能力 |
| 3级 | 筑基中期 | 法术能力有所增强 |
| 4级 | 筑基后期 | 灵智进一步开启，能使用更复杂的妖术 |
| 5级 | 结丹初期 | 丹田生妖丹，能力大幅提升 |
| 6级 | 结丹中期 | 能使用更高级的妖术 |
| 7级 | 结丹后期 | 开始渡劫化形，化形成功后进入化形期（相当于元婴期） |
| 8级 | 元婴初期 | 完全化去兽类模样，与人类修士无异 |
| 9级 | 元婴中期 | 接近化神期的修为 |
| 10级 | 元婴后期 | 非常接近化神初期，实力极强 |
| 11级及以上 | 化神期及以上 | 神通愈加强大 |

**补充说明**：妖兽达到七级（结丹后期）开始渡劫化形，化形成功后进入化形期，这一阶段不再以“X级妖兽”相称，而是与人类修士相同，以化神、炼虚、合体、大乘划分。

### 3.2 妖兽分类体系

*   **按境界划分**：妖兽（1-10级及以上）→ 化形妖修（7级化形后）
*   **按珍稀度划分**：
    - **普通妖兽**：常见类型，如双瞳鼠、追风兔等
    - **上古灵虫/异兽**：蛮荒时期留存，极其罕见，如六翼霜蚣、啼魂兽
    - **真灵级**：天地间至高无上的存在，如真龙、天凤、鲲鹏等
*   **按天赋划分**：不同妖兽拥有各异的天赋神通（如不灭之体、自愈之体等）
*   **按血统划分**：纯血妖兽（血统纯正）、混血妖兽（含不同族类血统）

## 四、符箓篇

### 4.1 符箓分类体系

符箓是修仙者将法术预先封印于符纸中以便随时释放的消耗品，按时间与灵感来源于中国道教传统，忘语在小说中将其系统化为修仙体系中不可或缺的辅助手段。：

| 类型 | 制作门槛 | 特性 | 代表符箓 |
| :--- | :--- | :--- | :--- |
| **普通符箓** | 筑基期以上即可绘制 | 纯粹一次性消耗品，封印单个法术，威力有限 | 火球符、防御符、风刃符等 |
| **符宝** | 结丹期以上修士炼制 | 封印自身法宝十分之一威能，可多次使用，越用越弱；有“伪法宝”之称 | 飞剑符宝（金光上人所制）、青冥针符宝（青易居士所制） |
| **真宝** | 高阶修士炼制 | 最顶级的符宝，封印法宝三分之一威能，一次性消耗品，仅血脉后人可用 | 火鸟真宝 |
| **灵符** | 元婴期以上绘制 | 可多次使用的变异符箓，材料稀有罕见 | 挪移符、化身符等 |

### 4.2 代表性符箓名录

- **飞剑符宝**：韩立入黄枫谷前核心依仗，凭借此宝轻松击杀觊觎自己的修士
- **青冥针符宝**：元婴老怪青易居士本命法宝青冥针炼成，犀利之极，虚天殿中赠予韩立
- **火鸟真宝**：一次性顶级符宝，威力惊人，可轻松灭杀四大血侍
- **挪移符**：大挪移令与上古传送阵配合，可超远距离安全传送
- **天罡符**：高阶防御符箓

### 4.3 韩立制符之路

韩立在炼气期便开始学习制符之术，这既是他赚取灵石的重要副业，也是保命手段的重要补充。随着修为提升，他逐步掌握了更高阶符箓的绘制技巧，从基础攻击符到符宝的收集与使用，再到后期与阵法配合的符阵联动，符箓在韩立的战斗体系中始终占有一席之地。


## 五、功法篇

### 5.1 功法流派分类

《凡人修仙传》中的功法体系庞大，按修炼流派可分为以下几大类（详见上一轮世界观大纲中的“修仙途径与功法”部分）：

| 流派 | 核心特征 | 代表性功法 | 代表性势力 |
| :--- | :--- | :--- | :--- |
| **道门** | 中正平和，门槛较低，流传最广 | 长春功、青元剑诀 | 黄枫谷、落云宗 |
| **魔道** | 霸道凶险，速成但易走火入魔 | 托天魔功、天都尸火 | 魔道六宗 |
| **佛门** | 驱邪避煞，克制魔道鬼道 | 明王诀、八灵尺功法 | 大晋佛宗 |
| **儒门** | 需浩然之气，与学识心性相关 | 儒家功法 | 大晋儒门 |
| **妖族** | 炼化横骨、化形渡劫 | 妖修功法 | 万妖谷 |
| **鬼道** | 修炼阴魂鬼力，不入轮回 | 鬼道功法 | 鬼灵门 |

功法按侧重类型还可划分为三类：**实战强化型**（附带神通威力巨大）、**修炼速成型**（法力提升快但战力偏弱）、**顶阶全能型**（集前两者优点但修炼条件苛刻风险大）。

### 5.2 韩立核心功法体系

韩立的功法体系是其越阶战斗和最终成为道祖的基石，按类型可分类如下：

**一、神识强化类（控场与感知核心）**

| 功法名称 | 来源/创作者 | 核心能力 | 战略价值 |

## 六、阵法篇

### 6.1 阵法分类体系

阵法在《凡人修仙传》中是洞府防御、宗门守护、战斗困敌的重要手段。按功能可分为：

*   **迷幻类**：掩饰山门、迷惑困敌（如颠倒五行阵、幻行天罗阵）
*   **攻击类**：主动攻击敌人（如天风狂烈阵、风炎龙烈阵）
*   **防御类**：守护洞府或城池（如四象玄武阵、大化白莲禁制）
*   **困敌类**：限制敌人行动（如小须弥金刚阵、六遁水波阵）
*   **辅助类**：聚集灵气、隔绝气息等（如拘灵阵、转轮聚阴阵）
*   **禁断类**：顶级大阵，化神修士也难以突破（如十方灭炎阵、木界三十六天绝阵）
*   **复合类**：兼具多重功能的大型阵法

### 6.2 代表性阵法名录

**人界著名阵法**：

- **颠倒五行阵**：号称“小禁断之阵”，可掩饰山门、迷惑困敌，由五种阵法构成，其中木之阵·竹海可将竹叶化为利器

- **四象阵**：经典困敌阵法
- **天风狂烈阵/幻行天罗阵**：范围较广，可使筑基期修士头疼
- **六遁水波阵**：需六人主持，既可困敌又可伤敌
- **太妙神禁**：十大古禁之一，出现在苍坤上人遗址处
- **血咒之门**：出现于坠魔谷，封印血焰主魂
- **小须弥金刚阵**：佛宗阵法，又叫金刚困仙阵，只能依靠蛮力破除
- **万木大阵**：以天桑神树为阵眼所布，上古奇阵之一
- **九真伏魔大阵**：人界辟邪退魔第一等厉害的法阵
- **冰魄寒烈阵**：由一千零八根万年玄冰柱子组成
- **十方灭炎阵**：禁断法阵，化神修士进入也无法马上破阵而出
- **八门引雷阵**：韩立自创，用于防止辟邪神雷反噬
- **转轮聚阴阵**：用于聚集阴气
- **御雷万封大阵**：设置在万灵台，为限制万灵

**灵界/仙界著名阵法**：

- **七星精火阵**：天渊城防御法阵，杀手锏之一
- **九阳罡日法阵**：倚天城防御阵法，吸纳天地阳力越多威能越强
- **木界三十六天绝阵**：模仿上界仙人玄天灵域大神通创立，威力奇大
- **八元锁魂阵**：真仙界阵法，用来封印螟虫之母

韩立在阵法之道上造诣颇深，尤其随着修为提升，阵法成为他安身立命的重要保障。从早期的颠倒五行阵守护洞府，到后期自创八门引雷阵防止辟邪神雷反噬，再到布置两仪万引阵守护元合岛，韩立对阵法的掌握贯穿了整个修仙历程。在他手中，阵法不仅是防御工具，更是配合法宝、符箓、灵宠实现越阶战斗的重要战术手段。


## 💎 六系总结

《凡人修仙传》之所以成为“凡人流”开山之作，很大程度上归功于其**法宝、灵丹、妖兽、符箓、功法、阵法**这六大体系的严谨设计与深度融合。它们之间并非孤立存在，而是一个环环相扣的有机整体。以主角韩立的成长路径为例，可以清晰地看到六大体系的协同运作：功法提供内功根基与战斗框架，法宝实现战斗力外化，灵丹突破瓶颈加速成长，妖兽作为战略伙伴辅助侦查与输出，符箓提供应急手段和越阶保命的可能，阵法守护洞府与战场控局。这六者共同构筑了一个逻辑自洽、令人沉浸的修仙宇宙，也正是这部作品能够跨越十余年连载、收获海量忠实读者的根本所在。
